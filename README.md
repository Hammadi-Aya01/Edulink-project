# 🎓 EduLink - Online Learning Platform

EduLink is a responsive educational website designed to provide an engaging learning experience. The platform showcases courses, instructors, and educational resources through a modern and user-friendly interface.

---

## ✨ Features

- Responsive design
- Home page
- About page
- Courses section
- Contact page
- Navigation menu
- Modern and clean UI

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
